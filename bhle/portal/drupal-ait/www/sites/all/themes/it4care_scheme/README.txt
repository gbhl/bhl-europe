bhleurope is a blank canvas for theme developers. 
Use bhleurope as a user friendly starting point to help facilitate your theme development. 
Build site themes without feeling like you have to reinvent the wheel or remove unnecessary code every time.

bhleurope 3 is intended to be used with the "Seven" administration theme included with Drupal 7.

USAGE

Install to sites/all/themes

To fully convert bhleurope to your own custom theme, it is encouraged that you: 
 - Copy the 'bhleurope' directory within sites/all/themes/ and rename it as 'yourthemename' 
 - Change the filename and contents of the .info file accordingly
 - Use find/replace in template.php to replace 'bhleurope' with 'yourthemename' 
 - Upload and enable the theme
 - Modify as needed

To create a sub-theme based on bhleurope, see: http://drupal.org/node/225125
